# MFD Duty Roster

Department of Hydrology & Meteorology, Nepal

## Setup
1. Deploy to Netlify by connecting this GitHub repository
2. Open the app → Admin Login
3. Go to Upload tab → paste your Google Apps Script URL → Save URL
4. Upload roster files for each role

## Editing Duty Names
Open `index.html` → search for `const DUTY_NAMES` → edit labels

## Editing Timings  
Open `index.html` → search for `const DUTY_TIMES` → edit timings

## Changing Script URL
Open `index.html` → search for `DEFAULT_SCRIPT_URL` → update URL
